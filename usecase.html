App Use Case

Growify AI is a social-media-management and analytics tool designed for creators and businesses who want to understand and improve their Instagram performance.

Purpose of the App

The app connects with the Instagram Graph API to retrieve basic account information, followers’ insights, and engagement metrics. These data help users track post performance and optimize content strategy.

Permissions Used

  • instagram_basic – to access account ID, username, and profile information.
  • pages_show_list – to link and display connected Facebook Pages.
  • instagram_manage_insights – to read insights and analytics for the user’s media.

Data Handling

No personal information is stored permanently. The data retrieved through the API is used only within the session to display analytics to the authenticated user.
